# Festival Assignment System

Technical MVP for validating the viability of a fair, consistent, and scalable festival seating assignment system.

## Current stage

**Stage 1 — Essential Domain**

The project is currently focused on understanding the business domain, defining its shared language, and identifying the rules that the assignment system must never violate.

No production-ready application is expected at this stage.

## Project status

Under active discovery and domain modeling.
